Программа реализует Марковский источник.

1. Запустить программу
2. При появления сообщения «Input n {2..6} = » ввести количество состояний марковской цепи. Число должно быть быть в интервале 1<k<7.
3. При появления сообщения «Do you want input P{}? [1 - yes, 'another' - no]:» введите «1» , если хотите ввести матрицу переходных вероятностей, и другое число  - если хотите, чтоб она сгенерировалась.

Выходом программы являются 2 файла: файл с сообщением(msg.txt) и файл с матрицей переходных вероятностей(matrix.txt).

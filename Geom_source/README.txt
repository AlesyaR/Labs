1. Запустить программу.
2. При появления сообщения «Input p» ввести число 0 < p < 1.
3. При появления сообщения «Input length of message :» ввести число равное количеству символов в сообщении
4. Выходом программы являются три файла:
	- msg.txt - сообщение
	- pr.txt - функция распределения, полученная практически
	- th.txt - функция распределения, полученная теоретически
	по двум последним файлам строится сравнения практической и теоретической функция распределения для проверки правильности генерации сообщения.
